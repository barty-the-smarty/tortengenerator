# Tortengenerator
Bild (Draufsicht) einer Geburtstagstorte mit einer frei wählbaren Anzahl von Tortenstücken.
Das generierte Bild kann zur Darstellung von Bruchrechnung verwendet werden. 

getestet mit Pyhton 3.11.9
benötigt MatPlotLib (pip install matplotlib)

- Variable NUM_SLICES steuert die Anzahl der Tortenstücke - bei Bruchrechnung der Nenner
- Variable NUM_LEAVES steuert die Anzahl der Blätter bei den Deko-Blüten (für die Berechnung nicht relevant, aber es soll auch hübsch aussehen :)
